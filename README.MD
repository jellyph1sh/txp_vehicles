# **TXP_VEHICLES**

## **DESCRIPTION :**
Basic commands to manage vehicles.

## **HOW TO CONFIGURE :**
- *Anythings able to be config are in `shared/config.lua`*

## **HOW TO USE :**
- *Command `/veh <model>` to appear in the vehicle of your choice.*
- *Command `/delveh` to delete vehicle you are in.*
